# Servo Motor Control using Potentiometer

## Project Description
This project demonstrates how to control a servo motor using a potentiometer
and Arduino. By rotating the potentiometer knob, the servo motor angle changes
accordingly.

## Components Used
- Arduino / ESP32
- Servo motor
- Potentiometer
- Jumper wires

## How the Project Works
- Potentiometer provides an analog input value
- Arduino reads the analog value
- The value is mapped to servo motor angle (0 to 180 degrees)
- Servo motor rotates based on potentiometer position

## Concepts Used
- Analog input
- Servo motor control
- map() function
- Arduino libraries

## Learning Outcomes
- Learned how to interface a servo motor
- Understood analog input using potentiometer
- Learned to control motor position
- Improved embedded programming skills

## How to Run
1. Connect potentiometer to A0 pin
2. Connect servo motor signal wire to pin 9
3. Upload the code to Arduino
4. Rotate the potentiometer and observe servo movement
